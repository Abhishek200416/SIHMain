# Results Directory

This directory should contain CSV files with hotspot analysis results.

## Required Files

### Hotspot Clusters
- `hotspot_clusters.csv`

### Site Evaluation Metrics
- `site_wise_evaluation_metrics.csv`

## How to Upload Results

1. Copy your CSV files from your local machine to this directory
2. Ensure file names match exactly as listed above
3. The application will automatically use these files when ML models are enabled

## Note

These files are used in conjunction with the ML models for comprehensive air quality analysis.
